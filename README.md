Легковесная утилита для мониторинга процессов в операционной системе Windows, разработанная на платформе WPF (C#). Проект создан в образовательных целях для изучения системного взаимодействия, работы с API операционной системы и построения интерфейсов на XAML.

## Основные возможности

* **Мониторинг процессов:** Вывод актуального списка запущенных в системе процессов.
* **Детальная информация:** Отображение ключевых метрик для каждого процесса: имя, идентификатор (PID) и объем занимаемой оперативной памяти.
* **Управление процессами:** Возможность принудительного завершения выбранных задач из интерфейса приложения.
* **Обновление данных:** Ручное обновление списка процессов для получения актуальных сведений из ОС.

## Стек технологий

* **Язык программирования:** C# (.NET)
* **Интерфейс:** WPF (Windows Presentation Foundation), XAML
* **Взаимодействие с ОС:** Пространство имен `System.Diagnostics`

## Инструкция по запуску

1. Склонируйте репозиторий:
```bash
git clone https://github.com/amadesya/WPFTaskManager
```

Откройте файл решения (.sln) в среде разработки Visual Studio или JetBrains Rider.

Выполните сборку и запуск проекта.

Примечание: Для получения доступа к некоторым системным процессам и их завершения может потребоваться запуск утилиты с правами Администратора.
